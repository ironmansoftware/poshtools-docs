# System Requirements

## Operating System

* Windows.

PSCommander uses Windows desktop integration points, WPF, Windows Forms, Explorer context menu registration, tray icons, file associations, custom protocol handlers, and global hot keys.

## PowerShell

* Windows PowerShell 5.1 or PowerShell 7.
* PowerShellGet or PSResourceGet configured for the PowerShell Gallery.

## Runtime

* .NET 6 Windows Desktop runtime support is required by the PSCommander desktop application.

## Permissions

Some features modify user-level Windows integration points. Per-machine changes or protected paths may require elevation.
