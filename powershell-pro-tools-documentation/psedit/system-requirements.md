# System Requirements

## PowerShell

* Windows PowerShell 5.1 or PowerShell 7.
* PowerShellGet or PSResourceGet configured for the PowerShell Gallery.

## Terminal

PSEdit runs in a terminal host. Use a terminal that supports interactive console applications.

## Optional Dependencies

* PSScriptAnalyzer is required for PowerShell formatting.

## Supported File Types

* PowerShell
* JSON
* YAML
* XML
* Markdown
* Plain text
